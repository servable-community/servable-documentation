# Anatomy of a protocol

Any protocol respects the following root folder structure:

```bash
┌── beforeinit
├── afterinit
├── functions
├── schema
├── config
├── class
├── classes
├── jobs
├── seed
├── lib
├── assets
├── triggers
├── documentation
├── liveclasses.js
└── index.json
```
